# inventory_crm_webapp
In the project terminal you can run 
mvn spring-boot:run 
Runs the app in the development mode.
Open http://localhost:8080 to view it in the browser.
<h3>Technologies and Tools</h3>
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>Javascript</li>
  <li>Bootstrap 5</li>
  <li>Thymeleaf</li>
  <li>PostgreSQL</li>
</ul>
